User Authentication & Profile Management System
This project includes a complete authentication flow with Login, Register, Sign In, and a fully functional Profile Page. The profile page allows users to upload images, add captions, and manage their profile details smoothly.
Features
 
1. Authentication
User Registration
User Login / Sign In
Input validation for secure authentication
Smooth UI flow between pages

2.Profile Page
Upload profile images
Add image captions
Display uploaded images in the profile
Clean and responsive design

3.Tech Stack
Frontend: HTML, CSS, JavaScript (or React if you're using it)
Backend: Node.js / Express (update if different)
Database: MongoDB / MySQL (update as needed)
